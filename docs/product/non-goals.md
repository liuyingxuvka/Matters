# Product non-goals

The first Matters release is not:

- a bulk Jira importer or Jira clone;
- a system that equates one issue with one Matter;
- an autonomous source of truth for facts, relationships, events, or status;
- a tool that treats Jira `Done`, a file named `final`, or a finished meeting as
  proof that a real-world matter is complete;
- a causal-discovery or treatment-effect system;
- an AI system that lets Guard findings or forecasts mutate canonical data;
- a UI-specific data model or a second set of English and Chinese facts;
- a cloud service that uploads private local data by default;
- a compatibility layer preserving multiple successful paths for retired
  fields or contracts;
- a public dataset derived by merely renaming private examples.

These boundaries may change only through a new OpenSpec requirement change and
the corresponding model, owner, test, privacy, and release evidence.
