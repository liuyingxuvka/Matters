# README hero design note

- **Product:** Matters, a local-first personal intelligence desktop application.
- **Audience:** people evaluating the public repository and Windows release.
- **Problem shown:** authorized personal sources are scattered and difficult to
  understand as coherent, changing Matters.
- **Workflow shown:** sources on the left, local understanding in the center,
  object-browser cards, stages, timeline, and people relationships on the right.
- **Privacy cue:** one protective local boundary surrounds every source and
  derived projection.
- **Visual language:** bright, mature, dimensional, navy/white/coral; no literal
  screenshot and no private data.
- **Asset:** `docs/assets/readme-hero/hero.png`.
- **Placement:** the single README hero immediately below the language links.
